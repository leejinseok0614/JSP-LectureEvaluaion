# JSP Lecture Evaluation

JSP 기반 강의평가 사이트 학습 프로젝트입니다.  
사용자가 강의를 평가하고 조회하는 서비스 흐름을 구현하면서 JSP, Bootstrap, jQuery, MySQL 기반 웹 애플리케이션 구조를 익히는 데 목적을 두었습니다.

## 주요 기능

- 강의평가 서비스 화면 구성
- Bootstrap 기반 반응형 UI
- jQuery를 활용한 동적 화면 처리
- JSP 기반 서버 사이드 렌더링
- MySQL 연동을 고려한 서비스 구조 학습

## 기술 스택

- Java
- JSP
- Bootstrap
- jQuery
- MySQL

## 프로젝트 구조

```text
src/main/webapp/
  index.jsp
  css/
  js/
```

## 학습 포인트

- JSP 프로젝트 기본 구조 이해
- 강의평가 도메인의 화면 흐름 설계
- Bootstrap을 활용한 빠른 UI 구성
- 서버 렌더링 기반 웹 서비스의 요청/응답 흐름 학습

## Reference

- 인프런 JSP 강의평가 사이트 강의를 참고하여 학습했습니다.
